# Spielplatz — German Article Trainer

An interactive drag-and-drop tool for practising German grammatical articles across Nominativ, Akkusativ, and Dativ cases.

## Getting started

No installation required. Open `german_article_trainer.html` directly in a browser.

## Features

- **Drag articles** (`der`, `die`, `das`, `ein`, `eine`, `den`, `einen`, `dem`, `einem`, `einer`) from the sidebar onto the canvas
- **Search nouns** by German name or English meaning and place them as blocks on the canvas
- **Pair articles with nouns** by dragging them close together — the pair turns green if correct, red if wrong
- **Draw case zones** (Akkusativ / Dativ) on the canvas to test declension — pairing correctness updates automatically based on which zone a noun sits in
- **Upload a `.txt` file** to extract nouns from your own text and scatter them on the canvas
- **Undo / Redo** with Ctrl+Z / Ctrl+Y
- **Multi-select** with marquee drag; delete selected blocks and zones with Delete/Backspace
- **Gender hints** (Settings ⚙) — toggle colour coding: blue = masculine, pink = feminine, green = neuter
